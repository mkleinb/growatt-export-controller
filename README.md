# Growatt Export Controller

Home Assistant custom integration for controlling Growatt export limits via the Growatt cloud.

## Features

- Set Growatt export limit from Home Assistant
- Toggle meter-based export control
- Home Assistant service: `growatt_export_controller.set_export_limit`
- HACS-ready structure

## Installation

1. Add this repository to HACS as a custom repository.
2. Install the integration.
3. Restart Home Assistant.
4. Add the integration from **Settings → Devices & Services**.

## Configuration

Configure your Growatt account and inverter details in the Home Assistant config flow.

## Support
